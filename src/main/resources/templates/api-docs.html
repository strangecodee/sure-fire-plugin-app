GET /api/result/{marks}

Check student result by marks. Returns JSON response with pass/fail status.

Path Parameters

Parameter Type Required Description
marks integer Yes Student marks (0-100)

Response Schema

{ "marks": "integer", "result": "string (Pass/Fail)", "message": "string" }
